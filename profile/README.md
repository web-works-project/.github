# web-works-project

## 🌐 概要
このOrganizationは、職業訓練校「これからはじめるデザイン・マーケティング科」（半年間）で学んだ内容を整理するためのリポジトリ群です。  
Web制作の基礎から応用までを体系的に学び、実際の成果物をポートフォリオとしてまとめています。

## 📖 学習内容
- HTML / CSS / JavaScript
- プロトタイプWebデザイン
- レスポンシブWebデザイン
- Illustrator / Photoshop
- WordPressによるサイト構築
